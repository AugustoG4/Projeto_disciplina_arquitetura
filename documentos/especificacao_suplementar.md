# Especificação Suplementar

## 1. Desempenho
- O sistema deve responder a ações do usuário em até 2 segundos.
- Suporte para pelo menos 500 usuários simultâneos.

## 2. Segurança
- Dados de pacientes criptografados (AES-256).
- Autenticação via senha forte e/ou 2FA.
- Acesso restrito por perfil.

## 3. Usabilidade
- Interface intuitiva e responsiva.
- Layout compatível com dispositivos móveis.

## 4. Confiabilidade
- Disponibilidade mínima de 99%.
- Backup automático diário.

## 5. Portabilidade
- Desenvolvido em plataforma web (HTML, CSS, JS, Python ou Node.js).
- Suporte a dispositivos móveis (design responsivo).

## 6. Conformidade Legal
- LGPD (Lei 13.709/2018)
- Normas éticas do CFP (Conselho Federal de Psicologia)
