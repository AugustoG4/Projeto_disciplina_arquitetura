# Protótipos das Principais Telas

## 1. Tela de Login
- Campos: e-mail, senha
- Botão “Entrar”
- Link “Esqueci minha senha”

## 2. Tela de Agenda
- Calendário interativo
- Botões “Agendar”, “Cancelar”
- Filtro por paciente e psicólogo

## 3. Tela de Atendimento
- Dados do paciente
- Campo para registrar observações
- Botão “Salvar no prontuário”

## 4. Tela de Pagamentos
- Lista de consultas e valores
- Status: Pago / Pendente
- Opção de gerar recibo

## 5. Tela de Lembretes
- Listagem de lembretes automáticos
- Botão para reenviar notificação

*(Protótipos visuais podem ser criados no Figma ou Draw.io e salvos em `docs/img/` como PNG ou SVG)*
